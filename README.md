# Mekonnen Anebo Personal Brand Site

A Vercel-ready Next.js personal brand website for Mekonnen Anebo.

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Lucide React icons

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Notes
- Social links, calendar link, and form handlers use placeholder values and are easy to swap.
- The contact form and newsletter form currently log submissions to the browser console.
- Main page content lives in `lib/site-data.ts` and `app/page.tsx`.
- Styling is centralized in `app/globals.css` and Tailwind config.
